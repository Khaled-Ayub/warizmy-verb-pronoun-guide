# Warizmy Akademie – WebApp

Dies ist das Frontend Projekt für die Warizmy Akademie. Hier entsteht die Lernplattform mit Kursen zu Schafi Fiqh, Arabisch, Qurʾan und Alltag, inklusive Vokabeltrainer, Video Lektionen und weiteren Funktionen.
