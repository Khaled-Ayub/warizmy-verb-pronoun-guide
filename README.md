# Warizmy Akademie – WebApp

Dies ist das Frontend Projekt für die Warizmy Akademie. Hier entsteht die Lernplattform mit Kursen zu Schafi Fiqh, Arabisch, Qurʾan und Alltag, inklusive Vokabeltrainer, Video Lektionen und weiteren Funktionen.

## Wie kann ich den Code bearbeiten

### 1. Direkt in Lovable arbeiten

Rufe das Lovable Projekt auf  
https://lovable.dev/projects/REPLACE_WITH_PROJECT_ID  
und arbeite dort mit Prompts oder dem integrierten Editor.  
Alle Änderungen werden automatisch in dieses Repository übernommen.

### 2. Lokal mit eigenem IDE arbeiten

Wenn du lieber lokal mit VS Code oder einem anderen Editor arbeitest, kannst du dieses Repository klonen und Änderungen wie gewohnt committen und pushen.  
Voraussetzung sind Node.js und npm, diese lassen sich zum Beispiel mit nvm installieren  
https://github.com/nvm-sh/nvm#installing-and-updating

Schritte für die lokale Entwicklung:

```sh
# 1. Repository klonen
git clone <YOUR_GIT_URL>

# 2. In das Projektverzeichnis wechseln
cd <YOUR_PROJECT_NAME>

# 3. Abhängigkeiten installieren
npm i

# 4. Entwicklungsserver starten
npm run dev
